User Search by Number or Carrier

{}

API Documentation

GET /search?q=<query> - Search user by number or carrier. Returns JSON.

Example:

GET /search?q=918xxxxxxxxx
Response:
{
  "Number": "918xxxxxxxxx",
  "Carrier": "Vodafone",
  "Name": "John Doe",
  ...
}

POST /user - Add a new user. Send JSON body with user details.

POST /user
Body:
{
  "Number": "918xxxxxxxxx",
  "Carrier": "Vodafone",
  "Name": "New User",
  ...
}
Response:
{
  "message": "User added",
  "newUser": {...}
}
